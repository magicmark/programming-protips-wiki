## Contributing

Contributions very welcome! To get started, clone and build the repo:

```
$ git clone <your fork> && cd programming-protips-wiki
$ make build
$ yarn dev
```

### Adding a new protip

To add a new tip, use the generator:

```
$ yarn plop
```
